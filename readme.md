# jquery-eqcol

> Equal effing columns with jQuery


## Install

Browserify

```
$ npm install --save jquery-eqcol
```

Bower

```
$ bower install --save jquery-eqcol
```

```html
<script src="bower_components/jquery-eqcol/dist/jquery-eqcol.min.js"></script>
```

## Usage

```js
import 'jquery-eqcol';

$('[data-eqcol]').eqcol({
	// options
});
```
